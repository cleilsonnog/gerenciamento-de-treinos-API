# Task 10: Painel Admin

## Descrição
Implementação do painel administrativo com rotas para gerenciar usuários, visualizar estatísticas e logs do Stripe.

## Rotas criadas
- `GET /admin/stats` — Estatísticas do dashboard (total usuários, por plano, assinaturas ativas, novos no mês)
- `GET /admin/users` — Lista usuários (paginado, busca por nome/email, filtro por plano/status)
- `GET /admin/users/:userId` — Detalhe do usuário
- `POST /admin/users/:userId/ban` — Banir usuário (usa BetterAuth admin plugin)
- `POST /admin/users/:userId/unban` — Desbanir usuário
- `GET /admin/stripe-logs` — Eventos do Stripe com filtros

## Alterações no schema
- `User`: adicionado `role`, `banned`, `banReason`, `banExpires`
- `Session`: adicionado `impersonatedBy`

## Plugins
- BetterAuth `admin()` plugin ativado com `ADMIN_USER_IDS` do env
