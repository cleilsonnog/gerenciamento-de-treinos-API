Plano: Integração ExerciseDB - GIFs de Exercícios

Contexto

Integrar a API ExerciseDB (v1, gratuita, sem API key) para exibir GIFs animados dos exercícios. Os exercícios do projeto estão em português, a ExerciseDB em inglês. Abordagem híbrida: admin
seleciona GIF ao criar exercício + AI busca automaticamente.

Decisões de Design

1.  Proxy no backend para ExerciseDB — evita CORS e permite cache futuro
2.  AI tool: após criar exercício, backend busca na ExerciseDB pelo nome em inglês (AI já conhece o nome em inglês)
3.  Exibição do GIF: thumbnail pequeno no card, clique abre modal com GIF maior

---

Etapas

1.  Backend — Prisma Schema

Arquivo: prisma/schema.prisma

- Adicionar gifUrl String? ao model WorkoutExercise
- Rodar npx prisma migrate dev

2.  Backend — Schemas Zod

Arquivo: src/schemas/index.ts

- Adicionar gifUrl: z.string().optional() nos schemas de body/response de exercícios

3.  Backend — Nova rota proxy ExerciseDB

Arquivo: src/routes/admin.ts

- GET /admin/exercises/search?q=bench+press → proxy para https://exercisedb.dev/api/v1/exercises/search

Arquivo: src/routes/ai.ts

- No tool createWorkoutPlan e addExerciseToWorkoutDay: após AI criar exercício, buscar na ExerciseDB pelo nome (que a AI já fornece em português, mas podemos pedir que forneça também nameEn)
